# MemoriaKids — 全体構想・システム概要・競合分析資料

## 1. エグゼクティブサマリー

**MemoriaKids** は、保育園における「登降園管理」「成長記録」「連絡帳」「写真サブスクリプション」を統合した **保育DXプラットフォーム** である。

### コアバリュープロポジション

> **「月500円で、かわいい子供の写真をたくさん届けたい」**

- 保護者の写真購入ペインポイントを **定額サブスク（月額500円）** で解消
- **On-Premise Edge AI** による顔認識で、わが子の写真を自動ピックアップ
- **特許取得済みの暗号化分散ストレージ** で、6年間の写真データを安全に保管
- **オープンソースモデル** によりエンジニアである保護者コミュニティが開発を支える

### プロジェクトゴール

| フェーズ | 期間 | 目標 |
|----------|------|------|
| **Phase 1: MVP** | 0〜6ヶ月 | 写真サブスク + 顔認識MVP。パイロット園 5園で実証 |
| **Phase 2: Growth** | 6〜18ヶ月 | 登降園・連絡帳・成長記録の統合。100園導入 |
| **Phase 3: Scale** | 18〜36ヶ月 | 全国展開。分散ストレージネットワーク構築。1,000園 |
| **Phase 4: Platform** | 36ヶ月〜 | 保育データプラットフォームとしてエコシステム形成 |

### KPI

- **ARR（年間経常収益）**: Phase 2 終了時 1億円（保護者 16,667人 × 6,000円/年）
- **導入園数**: Phase 3 終了時 1,000園
- **写真配信枚数**: 月間 100万枚（Phase 3）
- **顔認識精度**: mAP@0.5 ≥ 99.2%（園児特化 Fine-tuned モデル）
- **NPS**: ≥ 60（保護者）/ ≥ 50（保育士）

```
                    │  ③ データ分析・API (B2B2C)          │

                    │     → 売上の 10%                    │
                    │                                     │
                    └─────────────────────────────────────┘
```

### 3.2 分散コンピューティング還元モデル

```
保育園 PC が提供するリソース:
  ├── ストレージ (HDD/SSD) → 写真データの分散保管ノード
  └── GPU 計算能力 → 顔認識推論処理

還元の仕組み:
  ├── ストレージ提供量 × 稼働率 → 月額還元 (PC維持費に充当)
  └── GPU推論処理量 × 処理単価 → 月額還元 (マイニング方式)

計算例:
  ├── ストレージ 2TB 提供、稼働率99% → 月額 2,000円還元
  └── GPU (RTX 3060) 月間 10,000推論 → 月額 1,500円還元
  合計: PC1台あたり月額 約3,500円の還元 → PC購入費を約18ヶ月で回収
```
