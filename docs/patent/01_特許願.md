## 電子出願の手順

### 事前準備（初回のみ）

1. **電子証明書の取得**
   - マイナンバーカード（公的個人認証サービス）が利用可能
   - ICカードリーダーが必要

2. **電子出願ソフトのインストール**
   - 特許庁「インターネット出願ソフト」をダウンロード
   - https://www.pcinfo.jpo.go.jp/site/1_start/step-02.html
   - 動作環境: Windows 10/11、Java実行環境

3. **申請人利用登録**
   - 電子出願ソフトから「申請人利用登録」を行う
   - 識別番号が付与される（以後の出願で使用）

### 出願書類の変換

4. **本Markdownファイルを特許庁XML形式に変換**
   - 電子出願ソフトの「HTML変換」機能を使用
   - 又は、Word/一太郎形式で作成し、ソフト内で変換
   - 推奨: 各ファイルをWord(.docx)に変換してから取り込み

5. **図面の取り込み**
   - draw.ioで作成した図面をPNG（300dpi以上、白黒）でエクスポート
   - 電子出願ソフトに画像ファイルとして取り込み

### 出願実行

6. **書類チェック → 送信**
   - 電子出願ソフトで書類一式を読み込み
   - 「書類チェック」で形式エラーがないことを確認
   - 「出願」ボタンで特許庁に送信
   - 受領書が発行される（出願番号が確定）

### 出願後

7. **出願審査請求**（出願から3年以内）
   - 審査請求料: 138,000円 + 請求項数 × 4,000円
   - 個人出願の減免: 1/2〜1/3に軽減される場合あり
   - 本件（10請求項）の場合: 138,000 + 40,000 = 178,000円（減免前）
